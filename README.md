# TCS-Transport

### Projekt pierwszy ###
Autorzy: *_Jacek Markiewicz_* & *_Bartosz Bromblik_*

Projekt ten stanowi różne próby implementacji oraz wizualizacji algorytmów przedstawionych na zajęciach.
Wszystkie algorytmy i wizualizatory testowane były na mapie Krakowa.

# Zawartość repozytorium
- `visualiser/` - prosty wizualizator do algorytmu Dijkstry. Jako że była to pierwsza próba wizualizacji należy to traktować bardziej jako eksperyment mający na celu zaznajomienie się z narzędziami do wizualizacji. 
- `visualiser_alt/` - wizualizator do algorytmu ALT.
- `visualiser_arcflags/` - wizualizator do algorytmu Arc-Flags.
- `visualiser_astar/` - wizualizator do algorytmu A*.
- `visualiser_biarcflags/` - wizualizator do algorytmu Arc-Flags, wersja dwukierunkowa.
- `visualiser_bidijsktra/` - wizualizator do algorytmu Dijkstry, wersja dwukieronkowa.
