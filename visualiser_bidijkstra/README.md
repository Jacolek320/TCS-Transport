# Arc-Flags
Aby zkompilowańá:
1) `mkdir build && cd build`
2) `cmake .. && cmake --build .`

Aby uruchomińá:
1) `./PathVisualizer`
2) Kliknij LOAD i wybierz plik .osm.pbf
3) Kliknij na mapie na punkt by zaznaczyńá punkt startowy. NaciŇõnij drugi raz aby wybrańá koŇĄcowy.
4) NaciŇõnij START aby uruchomińá algorytm. PrńôdkoŇõńá moŇľna regulowańá odpowiednim suwakiem (uwaga: skala logarytmiczna). Przycisk PAUSE pauzuje symulacjńô. Przycisk RESET resetuje jńÖ.
